# 실행환경
본 모듈을 실행하려면 다음 환경이 갖춰져야 합니다.
* Anaconda Python 3.6 이상
* PhantomJS 실행환경
* Selenium

Anaconda 설치 파일은 [Anaconda Download](https://www.continuum.io/downloads)를 참조하시고 PhantomJS의 [다운로드](http://phantomjs.org/)는 본 링크를 참조하시기 바랍니다.

Selenium을 설치하려면

    pip install selenium

을 콘솔에서 실행합니다.
